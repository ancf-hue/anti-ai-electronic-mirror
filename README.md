# 🪞 Анти-ИИ Электронное зеркало

**Полностью офлайн. Приватно. Бесплатно.**

Приложение для отслеживания состояния кожи лица. Анализирует покраснение, сухость и отёчность, показывает динамику изменений относительно эталона.

---

## 🎯 Возможности

- 📏 **Точное определение расстояния** до камеры (landmarks 33 и 263 MediaPipe Face Mesh)
- 💡 **Измерение освещения** в люксах
- 🌡️ **Анализ покраснения** кожи
- 🧴 **Анализ сухости** кожи
- 💧 **Анализ отёчности** глаз
- 📊 **График динамики** изменений по дням
- 🔦 **Встроенный фонарик** для стабильного освещения
- 🔇 **Голосовые подсказки** (мужской синтез речи без интернета)
- 📄 **Экспорт отчёта** в TXT и JSON
- 🎨 **Смена темы** (тёмная / светлая / контрастная)
- 🔒 **100% офлайн** — все данные только на вашем устройстве

---

## 📋 Как использовать

1. Нажмите **«Запустить сканер»**
2. **Включите фонарик** (кнопка «🔦 ФОНАРИК»)
3. Встаньте на расстояние 30-40 см от камеры
4. Запомните числа **«💡 ОСВЕЩЕНИЕ»** и **«📏 ДИСТАНЦИЯ»**
5. Дождитесь зелёного света
6. Нажмите **«Сохранить замер»** (первый замер = эталон 0%)
7. При следующих замерах возвращайтесь к тем же числам см и лк

## ⚙️ Настройки (кнопка «⚙️ НАСТРОЙКИ»)

| Кнопка | Действие |
|--------|----------|
| 📄 Калибровка белым листом | Настройка камеры (один раз) |

| 🎨 Сменить тему | Тёмная / светлая / контрастная |
| 📄 Скачать отчёт | Резервная копия в TXT |
| 📦 Экспорт JSON | Полная копия данных |
| 🔇/🔊 Голос | Вкл/выкл подсказки |
| ❓ Как пользоваться | Инструкция |
| 🗑️ Очистить память | Удалить все данные |
